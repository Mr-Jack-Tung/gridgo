# gridgo

Website: [https://gridgo.io](https://gridgo.io)

A platform to create distributed systems more easier

### install

```xml
<dependency>
    <groupId>io.gridgo</groupId>
    <artifactId>gridgo-core</artifactId>
    <version>0.1.0</version>
</dependency>
```
